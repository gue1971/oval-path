# The Oval Path (prototype)

アメリカ大統領ノートアプリの初期プロトタイプです。

## 使い方

1. `oval-path/index.html` をブラウザで開く
2. セレクトで大統領を選ぶ
3. `Quick Insight` と `The Note` で背景とレガシーを確認

## 実装範囲

- モバイル縦長UI
- Image Frame（象徴アイテム）
- Quick Insight（名前、任期、政党、キーワード）
- The Note（Origin / Legacy）
- 思想の系譜マップ（初期版）
- 建国期8名のサンプルデータ

## 45人対応への段階案

- Phase 1: 1〜8代で文体・データ粒度を統一
- Phase 2: 9〜20代を追加し、時代別フィルターを実装
- Phase 3: 現代まで拡張し、検索・比較・お気に入り機能を追加

## 画像運用の現実案

- 第一選択: 象徴アイテム中心の統一イラスト
- 第二選択: AI生成でキャラ化し、NG時はパブリックドメイン肖像を色調統一して採用
- データ構造は `symbol` を `imageUrl` に差し替え可能
